# To get a random user `/random` [Get]

http://localhost:8080/api/v1/user/random

# To get all users `/all` [Get]

http://localhost:8080/api/v1/user/all

# Save an user `/save` [Post]

http://localhost:8080/api/v1/user/save

# Update an user `/update` [Patch]

http://localhost:8080/api/v1/user/update/?id=19

# Bulk-Update user `/bulk-update` [Patch]

http://localhost:8080/api/v1/user/bulk-update

# Delete an user `/delete` [Delete]

http://localhost:8080/api/v1/user/update/?id=19
